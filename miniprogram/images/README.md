# Tab Bar图标说明

请将以下图标放入 `miniprogram/images/` 目录：

## 需要准备的图标

### 1. edit.png / edit_active.png
- 用途：首页Tab图标
- 尺寸：81x81像素
- 描述：编辑/写作图标

### 2. history.png / history_active.png  
- 用途：历史Tab图标
- 尺寸：81x81像素
- 描述：历史记录图标

## 图标风格建议

- 保持简洁清晰
- 选中状态使用彩色（推荐蓝色渐变）
- 未选中状态使用灰色
- 圆角矩形背景
- 白色或浅色图标

## 示例图标获取方式

1. 使用图标网站（如IconFont、Flaticon等）搜索关键词：
   - "edit" 或 "write"
   - "history" 或 "clock"
2. 导出为PNG格式
3. 调整为81x81像素
4. 放入images目录
